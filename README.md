# Ping List

## Multiple url information gathering

This simple script allows you to ping a list of urls to gather information

## features

The script displays:

- URL
- Title
- Status (code and message)
- Server (if available)
- Content length

The data can be displayed and saved as:

- A table
- A list
- A markdown table
- As csv format
